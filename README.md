# Mobile Robot Simulation using Pygame

## Description

This project is a 2D mobile robot simulation built with Python and Pygame. It is created for learning, testing ideas, and experimenting with robot behavior inside a simulated environment.

The project is still under development. It is not finished yet.

## Current Status

This repository represents ongoing work. The structure may look unorganized and some parts of the code may be incomplete. Changes are expected as development continues.

You may notice unfinished features, temporary code, or areas that will be improved later. The main focus right now is building and testing functionality.

The project will be cleaned and structured better over time.

## Purpose

The main goals of this project are:

1. Simulate mobile robot movement in a 2D environment
2. Experiment with control logic and motion behavior
3. Build a base that can be extended in the future

## How to Run

1. Clone the repository:

   git clone [https://github.com/Altair098/Mobile-Robot-Simulation-_Pygame.git](https://github.com/Altair098/Mobile-Robot-Simulation-_Pygame.git)

2. Go to the project folder:

   cd Mobile-Robot-Simulation-_Pygame

3. Install required package:

   pip install pygame

4. Run the simulation:

   python main.py
